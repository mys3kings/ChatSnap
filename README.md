# Imperion - Secure Messaging App

A WhatsApp-style messaging app built with Firebase.

## Features
- Phone authentication
- Real-time messaging
- Contact management
- End-to-end encryption ready

## Setup
1. Clone this repo
2. Install Firebase CLI: `npm install -g firebase-tools`
3. Login: `firebase login`
4. Deploy: `npm run deploy`
