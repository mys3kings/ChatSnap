IMPERION v1.0 proposal

A sovereign messaging infrastructure · private, scalable, cross‑planet ready
Prepared: 26 Oct 2023 · Version 1.0 – confidential
for stakeholders & investors · 🔥 Firebase active

⚡ Imperion combines Erlang/OTP reliability, Signal Protocol privacy, and a unified cross‑interface experience to serve billions — individuals, SMBs, and enterprises.

📋 1. Product Requirements (PRD)

① Account & onboarding Phone verification, SMS 6‑digit, profile setup, permissions (contacts/camera).
② 1:1 / group messaging Text, media, read receipts, offline queue, group admin tools.
③ E2EE by default Signal Protocol, Curve25519, Double Ratchet. Media libs in Rust.
④ Voice/video calls Encrypted p2p & group calls, multi‑device.
⑤ Status & communities 24h stories, group clusters with announcements.
⑥ Chat Lock / vanish Biometric folder + disappearing 24h/7d/90d.
⑦ Business tools SMB app (catalog, auto replies) + Enterprise API + verified badge.
⑧ AI assistant Optional on‑device chatbot (blue circle).

⚙️ 2. Technical architecture

🧠 Backend core

  • ● Erlang/OTP – massive concurrency, hot code swap
  • ● FunXMPP‑like protocol – binary, tokens → messages as small as 20 bytes
  • ● Mnesia (in‑memory) for real‑time routing
  • ● Offline queue (Cassandra/Postgres) + push notifications
  • ● Media libs in Rust – memory safe, replace C++

🔐 Security & API

  • ● Signal Protocol (X3DH, double ratchet)
  • ● Client‑only key storage
  • ● Custom binary client API
  • ● REST/JSON Business API for enterprises
  • ● Perfect forward secrecy
💻 Desktop & multi-device: Web wrapper (WebView2) with QR linking from mobile. Two‑panel layout (chat list + conversation). RAM usage trade‑off accepted for faster feature parity.

🧭 3. User flows

Welcome→ Agree ToS→ 📱 Phone + SMS→ 6‑digit verify→ Profile (name/photo)→ Permissions→ Main app

Messaging flow – Alice encrypts → server checks Mnesia → online: direct delivery / offline: queue + push.

Business SMB flow – Download Imperion Business → register business phone → catalog & auto greetings.

🎨 4. UI/UX specifications

🟢 Login screens Country picker, phone, SMS, optional photo – clean, minimal.
⬇️ Bottom nav Chats · Calls · Communities · Status (floating translucent on iOS, same on Android).
💬 Chat list Contact photo, name, preview, time, unread badge; very round bubbles later.
🍏 Chat screen Green bubbles (me) / gray (others); attachment paperclip/+; paper airplane send.
⚙️ Settings Gear (iOS) or three dots (Android); profile, privacy, account.
🌙 Dark mode System wide, high contrast.
✅ unified design – Android adopts bottom bar, both platforms visually identical. Desktop two‑panel.
